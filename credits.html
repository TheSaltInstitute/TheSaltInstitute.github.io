This fansite is meant to represent the official website of the Magnus Institute, as it would have been in 1996... with a few extras
Launched as the Magnus Archives podcast comes to a close, it was made as part of the Eye Love TMA project, a collaborative effort to thank everyone involved in the creation of the podcast. We wanted to show you how much The Magnus Archives means to us and how much we appreciate all of the hard work you've put into it.

Sincerely, thank you.

Easter Eggs and privacy notes:
This website contains a number of Easter eggs — there should be one for every member of the cast and crew involved in the creation of the Magnus Archives. If this is you and you would like us to change or remove anything pertaining to you, or there are any problems at all, please tell us at magnuswebsiteproject@gmail.com!

  • This website is inaccessible to bots and should not show up in any search engine listings. If it does, something is broken; please let us know at magnuswebsiteproject@gmail.com so we can fix it.
  • We don't collect or store any data beyond what's required for website functionality. Details about what's collected where can be found here.

Technical notes:
The evolution of technology between 1996 and 2021 means that adblockers and other privacy extensions, as well as browser functions, can interfere with website functionality. Here are the things we know they break:

  • In some browsers, you will have to allow media autoplay to be able to hear the theme music on the index page.
  • If your adblocker is set to disable JavaScript, the scripts we used to display randomised content entries on the index page and the 404 page will not work, instead displaying every possible content entry at once.
  • Some extensions won't let you load the Entity "ad" banners in 'Our Generous Donors'.
  • The forums may be inaccessible with adblocker turned on.

Pinky promise that you can safely turn off your adblockers here. The website contains 0 cookies, 0 trackers, 0 actual ads, 0 malicious scripts — the only things getting blocked are the ones we listed above. Which is a shame, because many people put a lot of effort into making things for the website, and we want to make sure all contributions get seen!

When you're ready to peruse the Institute's website, click here!

For the best, most authentic 1996 navigation experience, we suggest viewing the website on a computer and making your browser window square-ish.
On mobile, some displays are very wonky, and we suggest switching to desktop view for a more user-friendly experience.
